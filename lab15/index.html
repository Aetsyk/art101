Lab 15 - AJAX

Challenge

The goal of this lab was to experiment with jQuery and AJAX and to print output from a different site's API to my site. I chose to use an API that gives random Jeopardy questions.

Problems

I wanted to have a button appear along with the question that would show the answer when clicked. However, for some reason, click listeners don't work with elements created by jQuery, or at least I couldn't get it to work. So I decided to create the button outside of the output and hide it, revealing it when the question is generated and hiding it when clicked or a new question is being fetched.
