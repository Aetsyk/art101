Lab 11 - JavaScript Events and Forms

Challenge

The challenge of this lab is to experiment with JavaScript events to create a program that sorts a string (and maybe make an anagram from it too).

Problems

I didn't have any major problems during this lab, expecially since I was allowed to reuse code from a previous lab. I keep forgetting to include the # when having jQuery pick out ids, and it takes me a bit to realize every time. My biggest challenge was implimenting the features I came up with for Task X, as a lot of it had me doing things that were unfamiliar to me, like clearing a section and evaluating input from multiple sources.
